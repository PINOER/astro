If you frequently use ChatGPT, Claude, or DeepSeek to write articles, summarize notes, or generate technical documentation, you've likely bumped into these frustrating formatting quirks:

* **Outer Code Fences**: The AI wraps the entire response inside ` ```markdown `, forcing you to manually delete the first and last lines every single time.
* **Inline HTML Artifacts**: Raw tags like `<span>`, `<br>`, or `<b>` pollute your text, breaking layouts when pasted into Word, WordPress, or Notion.
* **Excessive Line Breaks**: Generous spacing that leaves 3 or 4 empty lines between paragraphs, requiring tedious manual backspacing.
* **Inconsistent Spacing**: Cluttered punctuation and awkward whitespace that look unprofessional in published work.

## 💡 Key Features for Instant AI Text Cleanup

This tool was designed with a single goal: **transform messy AI outputs into clean, ready-to-publish Markdown or plain text in a single click**.

### 1. Strip Unwanted Inline HTML
When generating rich text or tables, LLMs often inject raw HTML tags. The cleaner automatically **strips out all inline HTML elements** while intelligently converting `<br>` and `</p>` tags into natural line breaks.

### 2. Compress Multiple Blank Lines
It detects consecutive line breaks and compresses them into standard paragraph spacing (leaving a maximum of one blank line), giving your content a clean, balanced rhythm.

### 3. Remove Outer Code Block Wrapping
It automatically identifies and strips away leading/trailing ` ```markdown ` or ` ``` ` code fences, leaving you with raw, clean content ready for use.

### 4. Trim Whitespace & Clean Margins
Eliminates unwanted leading and trailing spaces on every line, ensuring seamless copy-pasting into any editor.
